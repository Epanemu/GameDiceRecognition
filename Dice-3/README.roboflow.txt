
Dice - v3 better_augmentation
==============================

This dataset was exported via roboflow.ai on January 1, 2022 at 3:59 PM GMT

It includes 1059 images.
Dice are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Resize to 640x640 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* Randomly crop between 0 and 17 percent of the image
* Random brigthness adjustment of between -15 and +15 percent
* Random Gaussian blur of between 0 and 2 pixels
* Salt and pepper noise was applied to 1 percent of pixels
